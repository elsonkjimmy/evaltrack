## 🚀 About EvalTrack

EvalTrack is a comprehensive evaluation tracking system designed to streamline assessment processes, feedback collection, and performance monitoring. Built with modern web technologies, it provides a solid foundation for building evaluation and tracking applications across various domains including education, HR, and project management.

### Key Features

- 📊 **Evaluation Dashboard** - Centralized view of all evaluations and assessments
- 📝 **Custom Evaluation Forms** - Create and customize evaluation templates
- 📈 **Progress Tracking** - Monitor progress and performance over time
- 🔔 **Real-time Notifications** - Stay updated on evaluation status changes
- 👥 **Multi-user Support** - Collaborative evaluation processes
- 📱 **Responsive Design** - Works seamlessly on all devices

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Modern web browser

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/elsonkjimmy/evaltrack.git
cd evaltrack
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Start the development server**
```bash
npm run dev
# or
yarn dev
```

4. **Open in browser**
Navigate to `http://localhost:5173`
